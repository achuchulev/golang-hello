# GoLang program that prints "Hello world"

### Prerequisites

* git
* GoLang

## Run program

```
git clone https://github.com/achuchulev/hello-go.git
cd hello-go
go run main.go
```

Build program into binary:

`go build  main.go`

Execute the built binary directly: 

`./main`

The program should produce a "*_Hello world_*" message
